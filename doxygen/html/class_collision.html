private

This method handles the rest of the collision checking after a collision is found.

When a collision is found on the main collisionBox this function checks where the collision occured. Possibilities are: Left, Right, Top and Bottom.
After checking which side the collision is detected the object is added to the corresponding vector. The collisionSides struct is updated with a true on the corresponding side.

Parameters
object1The first object to check the collision width.
object2The second object to check the collision width.
topVector with top collisioned objects.
bottomVector with bottom collisioned objects.
leftVector with left collisioned objects.
rightVector with right collisioned objects.
collisionSidesThe collisionSides struct
