Detailed Description

This is the Character class.

This class is used to make non playable characters and players for the game. The methods shared among all the characters in the game are in this class.

◆ animate()

void Character::animate ( const sf::Time &  deltaTime)

This method is used to animate the Character his movement.

This method is used to show the Character's animation on the screen. This is done using a vector containing std::pairs, this vector is called mAnimations. The vector is filled using bindAnimation and bindAnimations. In this vector the first item of the pair is the state and the second the AnimationSequence that should be displayed in that state.

Parameters
deltaTimeThe amount of time between the past update and this update. This is used to check whether a new picture should be selected.
